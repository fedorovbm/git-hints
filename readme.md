# git-hints

`git-hints` — краткий справочник по самым базовым и часто используемым командам Git. Репозиторий предназначен для быстрого ориентирования в повседневной работе с системой контроля версий.

## Базовые команды

### Инициализация и получение репозитория
```bash
git init                 # создать новый репозиторий
git clone <url>          # клонировать удалённый репозиторий
```

### Проверка состояния и история
```bash
git status               # текущее состояние файлов
git log --v              # краткая история коммитов
```

### Работа с изменениями
```bash
git add <file>           # добавить файл в индекс
git add .                # добавить все изменения
git commit -m "message"  # создать коммит
git push                 # отправить изменения в удалённый репозиторий
```
